# EazyPass Study

## ✍️ 자격증 수험생을 위한 모바일 학습 앱

EazyPass Study는 자격증 수험생을 위한 효율적인 학습 플랫폼입니다.  
- 퀴즈 풀기
- 기출 분석
- 오답 복습
- 이론 요약  
등 다양한 기능을 통해 짧은 시간 안에 집중 학습을 가능하게 합니다.

## 🔐 카카오 로그인 연동

앱은 카카오 로그인 기능을 통해 간편한 회원가입 및 로그인을 제공합니다.  
로그인 이후에는 사용자 개인별 **진도율**, **포인트**, **오답노트** 등의  
상태가 자동으로 저장되고 불러와집니다.
